# 產品需求文件 (PRD)

## 1. 產品定位

**個人使用、零成本的選股工具**,跑在本機或免費雲端,透過手機瀏覽器使用。

**不是商業產品**:沒有使用者管理、沒有訂閱、沒有上架計畫。

## 2. 為什麼選 Streamlit 而非 App

| 比較項目 | 手機 App (Flutter/RN) | Streamlit Web |
|---------|---------------------|---------------|
| 開發時間 | 1–2 個月 | 1–2 週 |
| 上架成本 | Apple USD 99/年 | 0 元 |
| 改邏輯後重新部署 | 重新編譯、安裝 | 存檔即更新 |
| 圖表函式庫 | 需挑選 fl_chart 等 | plotly 直接用 |
| 個人使用體驗 | 流暢但成本高 | 加到主畫面後幾乎無差 |

**結論**:個人用 Streamlit 是最佳解。

## 3. 使用者情境

### 3.1 日常使用流程

1. **下班後 17:00**:打開手機,點主畫面圖示進入 Dashboard
2. **看「今日短線推薦」**:5–10 檔符合短線條件的股票,附當日 K 線
3. **點任一檔股票**:看詳細指標(KD、MACD、籌碼面)
4. **每月一次**:切到「長線選股」頁,檢查口袋名單變化

### 3.2 替代流程(懶得開介面時)

- 排程每天收盤後自動跑,結果 push 到 LINE
- 直接在 LINE 訊息中看到「今日符合 XXX 策略的 5 檔股票」

## 4. 功能規格

### 4.1 必做 (Must-have)

#### F1. 資料抓取與快取
- **F1.1**:從 FinMind 抓台股每日收盤價、成交量
- **F1.2**:從 FinMind 抓三大法人買賣超、融資融券
- **F1.3**:從 FinMind 抓財報資料(EPS、ROE、營收)
- **F1.4**:從 yfinance 抓美股日線資料
- **F1.5**:所有資料儲存至 SQLite,只增量抓取新資料

#### F2. 技術指標計算
- **F2.1**:5/10/20/60 日均線
- **F2.2**:KD 隨機指標
- **F2.3**:MACD
- **F2.4**:RSI
- **F2.5**:布林通道

#### F3. 短線選股
- **F3.1**:預設策略「量價突破 + 法人買超」
- **F3.2**:輸出符合條件的股票清單
- **F3.3**:支援使用者調整關鍵參數(均量倍數、買超天數等)

#### F4. 長線選股
- **F4.1**:預設策略「高 ROE + 低 PE + 連續配息」
- **F4.2**:輸出符合條件的股票清單,附 5 年財務摘要
- **F4.3**:支援使用者調整門檻

#### F5. Streamlit 介面
- **F5.1**:首頁顯示「今日短線推薦」與「長線口袋名單」
- **F5.2**:個股詳情頁 (K 線圖 + 指標 + 籌碼)
- **F5.3**:策略參數可在 sidebar 調整,即時更新結果
- **F5.4**:支援台股 / 美股切換

### 4.2 選做 (Nice-to-have)

#### G1. LINE Notify 通知
- 每日收盤後排程跑,結果 push 到 LINE 群組

#### G2. 自選股清單
- 使用者標記關注股票,首頁顯示其當日表現

#### G3. 簡易回測
- 給定策略,回算過去 1 年的選股結果與報酬

#### G4. 大盤情緒指標
- 顯示加權指數、VIX、外資買賣超總額

### 4.3 不做 (Out-of-scope)

- 即時行情(分鐘級以下)
- 自動下單 / 券商串接
- 多使用者、登入、訂閱
- 加密貨幣、外匯
- 個股新聞、研報全文(版權問題)

## 5. 非功能需求

- **效能**:首頁載入 < 3 秒(查快取);資料更新 < 30 秒(抓 API)。
- **可用性**:在 iPhone Safari 加到主畫面後,操作體驗類似 App。
- **資料新鮮度**:台股當日收盤後 30 分鐘內可看到當日資料。
- **可維護性**:選股邏輯獨立成函式,方便日後新增策略。

## 6. 環境與相依套件

- Python 3.11+
- 主要套件:streamlit, finmind, yfinance, plotly, pandas, numpy, pandas-ta(或自己算指標)
- SQLite(Python 內建)
- 開發工具:pytest, ruff, uv(可選)

## 7. 風險與限制

- FinMind 免費版每小時 1500 次請求,需善用快取避免超量。
- yfinance 是非官方套件,Yahoo 改動會影響穩定性,但個人使用衝擊有限。
- 選股邏輯不保證獲利,介面需有風險警語。
